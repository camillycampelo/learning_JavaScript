## 📂 Estrutura do repositório

Atualmente, o repositório contém:

```

learning_JavaScript/
├── 02.js
├── browser.js
├── index.html
├── index.js
├── package.json
├── package-lock.json
└── tempCodeRunnerFile.js

````

- `02.js`: exercício ou implementação relacionada ao módulo “02” do curso.
- `browser.js`: código para testar em ambiente de navegador ou manipulação de API do browser.
- `index.html`: página de teste para interagir com JavaScript no navegador.
- `index.js`: lógica principal conectada ao HTML.
- `tempCodeRunnerFile.js`: código auxiliar ou temporário usado em testes.
- `package.json` / `package-lock.json`: dependências e configuração do projeto.

## 🛠️ Como rodar / testar

Para ver os exemplos no navegador:

1. Abra o arquivo `index.html` no navegador.
2. Verifique no console (DevTools) os logs e resultados dos scripts.
3. Edite os arquivos `.js` e recarregue a página para testar mudanças.

Se quiser usar um servidor local (útil para testes com módulos, import/export, fetch etc.), você pode usar:

```bash
# exemplo com npm (se tiver instalado)
npx live-server
````

ou outra forma simples de servir a pasta atual.
